Setup 2: Installing Visual Studio Code

Editing and VisualStudio Code

When programming you write code using a text editor.

Notepad or others are examples of a text editor. Some editors have features that help programmer.

These features include running your code and looking for problems in code (called de-bugging).

Visual Studio Code is a tool for doing this you can find it here and install it.

Download for Windows Download for Mac OS

When installing select the 'Launch Visual Studio Code' options. Or run it after it is installed.

You can close it once you know it runs.
