Assignment 1: Modify Pick a Number

This assignment you will modify the pick an number program to:

  • Allow you to play games that are configurable on the maximum number 1-10, 1-20, 1-100 etc.
  • Change it so that the screct number is always the same until guessed and each turn checks that number
  • Change it so the 'Too high' and 'too low' say 'Out of range: Too Low' and 'Out of range: Too High'
  • Add a 'Guess: Too Low' and 'Guess: Too High' and make the this low/high to the secret

Hint 1: Constructor functions can take data

You can pass data to a constructor. This data is set when you call new.

In the below example 20 is sent as the value max.

    . . .
    constructor(max) {
        this.max = max;
    }
    . . .
    let game = new Game(20);
